**macOS menu bar app for monitoring TCP ports, processes, and Docker containers.**

## Download

👉 **[Download Latest Release](https://github.com/kudr9tov/portpal-releases/releases/latest)**

## Features

- 🔍 Real-time TCP port monitoring
- 🎯 Framework detection with 37+ icons (Spring, Node.js, React, Django, Flask, and more)
- 🐳 Docker container management (start/stop)
- 📂 Process categorization (Backend, Frontend, Database, DevTools, System)
- 🔎 IDE overlay icon detection (IntelliJ IDEA, VS Code, Xcode)
- ⚡ Kill processes with confirmation dialog
- 🔄 Auto-refresh with configurable interval
- 🔍 Search and filter

## Installation

1. Download `PortPal.dmg` from [Releases](https://github.com/kudr9tov/portpal-releases/releases/latest)
2. Open the DMG
3. Drag **PortPal** to **Applications**
4. Launch from Applications

## System Requirements

- macOS 14.0 (Sonoma) or later
- Universal Binary: Apple Silicon + Intel
